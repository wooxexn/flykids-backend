# FlyKids Backend 🛩️

이 저장소는 **FlyKids** 서비스의 백엔드 API 서버입니다.  
FlyKids는 초등학생을 위한 **태블릿 기반 드론 조작 교육 서비스**입니다.

---

## 📦 기술 스택

- Java 17
- Spring Boot 3.5
- Spring Security + JWT
- Spring Data JPA
- PostgreSQL
- Swagger (SpringDoc OpenAPI)
